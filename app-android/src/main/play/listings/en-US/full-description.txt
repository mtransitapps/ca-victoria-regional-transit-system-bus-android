This app adds Victoria Regional Transit System (BC Transit) buses information to MonTransit.

The app contains the planned schedule as well as real time predictions from NextRide and news from @BCTransit on Twitter.

Victoria Regional Transit System buses serve Victoria in British Columbia, Canada.

Once this application is installed, the MonTransit app will display buses information (schedule...).

This application only has a temporary icon: download the MonTransit app (free) in the "More ..." section bellow or by following this Google Play link https://bit.ly/MonTransitPlay

You can install this application on the SD card but it is not recommended.

The information comes from the GTFS file provided by BC Transit.
https://www.bctransit.com/open-data

This application is free and open-source:
https://github.com/mtransitapps/ca-victoria-regional-transit-system-bus-android

This app is not related with BC Transit, Victoria Regional Transit System and Victoria Regional Transit Commission.

Permissions:
- Other: required for real-time schedule predictions from nextride.victoria.bctransit.com and to read news from Twitter
